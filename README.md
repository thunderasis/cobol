# Demo de Modernización: De Legacy a Cloud Native
Este repositorio contiene los artefactos de prueba para la evaluación de la plataforma **Blitzy**.

## Escenarios de Prueba
Hemos preparado tres casos de uso críticos para la organización:

1.  **COBOL a Python:** Migración de lógica financiera (Cálculo de interés) de un monolito a un script mantenible.
2.  **Java Legacy a Modern Java:** Reducción de código verboso (Java 8) a estructuras modernas (Java 21 Records).
3.  **DataStage a Databricks:** Transformación de procesos ETL visuales a código PySpark optimizado para el Lakehouse.

## Estructura del Proyecto
* `calculo_interes.cbl`: Código fuente original en COBOL.
* `calculo_interes.py`: Propuesta de modernización en Python.
* `migracion_datastage.py`: Ejemplo de transformación de lógica ETL a Spark.
